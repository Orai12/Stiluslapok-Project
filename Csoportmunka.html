Ikonok hozzáadása A legegyszerűbb módja annak, hogy egy ikont hozzáadj a HTML oldalához, egy ikon könyvtárral, például a Font Awesome eszközzel.Adja hozzá a megadott ikonosztály nevét bármely inline HTML elemhez (például vagy ).Az alábbi ikonkönyvtárakban lévő ikonok skálázható vektorok, amelyek testreszabhatók CSS-sel (méret, szín, árnyék stb.). Fontos félelmetes ikonok A Font Awesome ikonok használatához adja hozzá a következő sort a HTML oldal szakaszában:link rel = "stíluslap" href = "https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css" Megjegyzés: Nincs letöltés vagy telepítésre szükség!
