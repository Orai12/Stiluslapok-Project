A 2D-s transzformációkkal színesebbé tehetjük weblapjainkat egy kis életet vihetünk bele. Maga a felhasználása nem igényel túl nagy tudást. Egész egyszerő 2D-s alakzatokat Css nyelvbe is meg tudunk valósítani, ha valami komlyabbat szeretnénk ahoz JavaScript szükséges.Az egészet divbe érdemes írni így minnél több van a weblapon annál egyszerűbb formázni őket. Igy ráadásul minden oldalunkra be tudjuk tenni egy egyszerű css fájlba, szóval ha minden oldalra alkalmazni szeretnénk akkor ez a legegyszerűbb módja. translate() rotate() scale() skewX() skewY() matrix() Most pedig nézzük meg hogy is néz ki ez egybe beírva:

div

{

width: 300px;

height: 100px;

background-color: yellow;

border: 1px solid black;

translate(50px,100px);

transform: translate(50px,100px);

}

Az összes többinél a bemutatóba leírtakat kell cserélgetni.
