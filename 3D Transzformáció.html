A 3D-s transzformációkkal színesebbé tehetjük weblapjainkat egy kis életet vihetünk bele. Maga a felhasználása nem igényel túl nagy tudást. Egész egyszerő 3D-s alakzatokat Css nyelvbe is meg tudunk valósítani, ha valami komlyabbat szeretnénk ahoz JavaScript szükséges.Magát a transzformációt rotálhatjuk X-Y-z tengelyen.Az egészet divbe érdemes írni, így minnél több van a weblapon annál egyszerűbb formázni őket. Igy ráadásul minden oldalunkra be tudjuk tenni egy egyszerű css fájlba, szóval ha minden oldalra alkalmazni szeretnénk akkor ez a legegyszerűbb módja.

transform: rotateX(150deg)

transform: rotateY(150deg)

transform: rotateZ(90deg)

Most pedig nézzük meg hogy is néz ki ez egybe beírva:

>div

{

width: 300px;

height: 100px;

background-color: yellow;

border: 1px solid black;

}

div#myDiv

{

transform: rotateX(150deg);

}

Ez az X-tengelyes rotáció.

 Az összes többinél a bemutatóba leírtakat kell cserélgetni.
